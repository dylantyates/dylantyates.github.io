# Dylan Yates

Personal static site — virtual presence and links. Built with [Astro](https://astro.build), version **1.0.0**.

## Local development

```bash
npm install
npm run dev
```

Open [http://localhost:4321](http://localhost:4321).

## Build

```bash
npm run build
```

Output is in `dist/`. Preview with `npm run preview`.

## Deployment (GitHub Pages)

This repo is set up for GitHub Pages. Pushing to `main` triggers a workflow that builds the Astro site and deploys it. Custom domain is configured via `public/CNAME` (dylantyates.com).
